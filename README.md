# ginasio-landing-page

Landing Page – Ginásio Renascimento

Este projeto é uma landing page de um ginásio criada para apresentar serviços, planos e incentivar novos alunos a se inscreverem.
O site foi desenvolvido com foco em design moderno, organização de conteúdo e experiência do usuário.

---

Sobre o projeto

A página foi construída como uma landing page promocional, onde o visitante pode:

- Conhecer o ginásio
- Entender a proposta e os valores do treino
- Visualizar benefícios
- Ver os planos disponíveis
- Interagir com os elementos da página

O objetivo principal do projeto é praticar desenvolvimento web e servir como parte do meu portfólio como desenvolvedor.

---

Funcionalidades da página

O site possui as seguintes seções:

Apresentação

Uma seção inicial com o nome do ginásio, imagem e chamada principal para atrair a atenção do visitante.

Motivação

Frases que representam os valores do ginásio:

- Transformação
- Disciplina
- Evolução

Banner principal

Uma imagem destacando a ideia de evolução constante no treino.

Sobre o ginásio

Explicação sobre a proposta do ginásio, destacando:

- disciplina
- força
- mentalidade
- evolução pessoal

Benefícios

Uma seção mostrando motivos para frequentar o ginásio.

Planos de treino

Apresentação de planos com preços fictícios:

- Plano semanal
- Plano mensal
- Plano anual

Chamada final

Mensagem motivacional para incentivar o usuário a começar a treinar.
